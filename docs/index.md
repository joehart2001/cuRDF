# cuRDF

GPU-accelerated radial distribution functions using Toolkit-Ops + PyTorch, with adapters for ASE (XYZ/extxyz/.traj) and MDAnalysis trajectories, supporting same- and cross-species RDFs.

- Quickstart: basic install and first RDF
- API: core functions and CLI flags

Install:
```
pip install -e .
```
